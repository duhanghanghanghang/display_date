# 我的衣柜 - 微信小程序

衣柜管理小程序，支持拍照保存衣服、分类管理、多维度筛选。

## 功能

- **衣柜首页**：分类统计、添加衣服、分类管理、全部衣服
- **分类管理**：增删改分类，点击进入该分类列表
- **衣服列表**：按分类/颜色/季节筛选，上拉加载、下拉刷新
- **添加/编辑**：拍照或选图上传，填写名称、颜色、尺码、季节等
- **衣服详情**：查看、编辑、删除

## 使用

1. 用微信开发者工具打开项目
2. 在 `project.config.json` 填写 AppID
3. 在微信公众平台将 `https://dhlhy.cn` 加入 request 合法域名
4. 编译运行

## 配置

API 基址在 `config/env.js`，默认 `https://dhlhy.cn`。
